Buy And Sell PHP script allowing you to open your own online store a Members can submit products, to which you can approve and reject. From then, other members can purchase these products, and you can get a cut of this, which can be defined by you in the Admin panel. This script contains a Secure download section, allowing only people who have downloaded the files to be able to download them, a fully working Membership system, categories of products, information on such products with comments and an Admin section.
